# 计算机组成原理
## 计算机原码，反码，补码
### 原码：
	是最简单的机器数表示法。用最高位表示符号位，‘1’表示负号，‘0’表示正号。其他位存放该数的二进制的绝对值。
	若以带符号位的四位二进值数为例： 
	    1010 ： 最高位为‘1’,表示这是一个负数，其他三位为‘010’，
	    即（0*2^2）+（1*2^1）+（0*2^0）=2（‘^’表示幂运算符）
	    所以1010表示十进制数（-2）。
### 反码：
	正数的反码还是等于原码；
	负数的反码就是他的原码除符号位外，按位取反。      
    若以带符号位的四位二进制数为例：
        3是正数，反码与原码相同，则可以表示为0011
        -3的原码是1011，符号位保持不变，低三位（011）按位取反得（100）
        所以-3的反码为1100
### 补码：
	正数的补码等于他的原码，
	负数的补码等于反码+1。
