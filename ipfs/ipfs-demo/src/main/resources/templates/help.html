Ruoli-API

根据Hash值可以查看位于区块链上的数据，例如：
https://ipfs.io/ipfs/QmRguPCYJZVGSFtntNospGuZWTHLSV93LXEDTDoBApYyVX/

GET 读取IPFS数据

http://ruoli.ink/ipfs/QmRguPCYJZVGSFtntNospGuZWTHLSV93LXEDTDoBApYyVX

返回读取的状态和值（status值为1时读取成功,result为读取出来的结果数据;值为-1时读取失败，result为错误信息）

Example Request
读取IPFS数据
 
										curl --location --request GET "http://ruoli.ink/ipfs/QmRguPCYJZVGSFtntNospGuZWTHLSV93LXEDTDoBApYyVX"
Example Response200 OK 
										{
  "status": 1,
  "result": "ruoli.ink"
}

POST 写入IPFS数据

http://ruoli.ink/ipfs/

返回对应IPFS区块链公链的Hash值,客户端需要保存此Hash值用于再次查询数据（status值为1时写入成功,值为-1时写入失败）

Body

formdata
recode ipfs-data
Example Request
写入IPFS数据
 
										curl --location --request POST "http://ruoli.ink/ipfs/" \
  --form "recode=ipfs-data"
Example Response200 OK 
										{
  "status": 1,
  "hash": "QmZRP7mdZ8DiW6k7tqCAft8PZy86krGm71dhvnm9TJZ4Nk"
}
